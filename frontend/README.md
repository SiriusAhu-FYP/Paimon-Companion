- `docs/TEST_SCRIPTS.md`：测试脚本文档
- `docs/测试脚本使用说明.md`：测试脚本简版快速上手

## 快速开始

```bash
pnpm install
pnpm dev
```

开发端口默认为 `http://localhost:7788`，若端口冲突可自行调整并通过 `LIVE2D_API_BASE` 或 `--port` 参数告知测试脚本。
